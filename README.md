# VIB Financial Performance Analysis Dashboard (2019–2025)

## Project Overview

This project analyzes VIB Bank's financial performance from 2019 to 2025 using publicly available annual report data.

## Data Source

The dataset used for this analysis is sourced from VIB Bank's publicly available annual reports (2019–2025). 
- [Dataset Link](https://www.cophieu68.vn/quote/financial_detail.php?id=vib&amp;type=year#google_vignette)

## Tools

- Power BI
- Power Query
- Excel

## Key Metrics

- Total Assets
- Revenue
- Customer Deposits
- Pre-tax Profit
- Credit Growth

## Key Insights

- Total Assets increased by 20%.
- Customer Deposits increased by 17%.
- Revenue decreased by 7%.
- Pre-tax Profit decreased by 16%.
- Credit Growth reached 22% during the post-COVID recovery period.
